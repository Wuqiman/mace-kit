Feature List
-------------

- [x] Face Detection
- [x] Face Recognition
- [x] Speech Recognition
- [x] Object Detection

How to build
-------------

Build with Android NDK
```sh
tool/cmake-build-android-armeabi-v7a.sh
```

Test on android
```bash
tool/cctest-android.sh
```

For more information, read the documentation.
